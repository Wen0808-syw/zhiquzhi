## 🅰️ 方案一：Vercel 部署（⭐ 推荐，最简单）

### 第1步：注册必要账号

#### 1.1 注册 GitHub 账号（免费）

> GitHub 是存放代码的地方，类似网盘

1. 打开浏览器，访问：**https://github.com**
2. 点击 **Sign up**（注册）
3. 填写信息：
   - **Email**: 你的邮箱
   - **Password**: 设置密码（建议 8 位以上）
   - **Username**: 起个英文名（记住它，后面要用）
4. 验证邮箱（去邮箱点击链接）
5. ✅ 注册完成！

#### 1.2 注册 Vercel 账号（免费）

> Vercel 是让网站上线的地方

1. 打开浏览器，访问：**https://vercel.com**
2. 点击 **Sign Up**
3. 选择 **Continue with GitHub**（用 GitHub 账号登录，最简单）
4. 授权确认 ✅

---

### 第2步：将代码上传到 GitHub

#### 2.1 创建代码仓库

1. 登录 GitHub：https://github.com
2. 点击右上角 **+** 号 → **New repository**
3. 填写信息：
   - **Repository name**: `zhiquzhi`（仓库名）
   - **Description**: `织趣织 - 编织爱好者社区平台`
   - **Private/Public**: 选择 **Public**（公开）
   - ✅ 不要勾选任何初始化选项！
4. 点击 **Create repository**

#### 2.2 上传项目代码

**方法 A：用网页上传（最简单）**

1. 在你刚创建的仓库页面
2. 点击 **uploading an existing file**（上传已有文件）
3. 打开电脑上的文件夹 `C:\Users\Administrator\WorkBuddy\20260416091656\zhiquzhi`
4. **全选所有文件和文件夹**（Ctrl+A）
5. 拖拽到网页上传区域
6. 滚动到页面底部，点击 **Commit changes**

**方法 B：用 Git 命令上传（推荐，但需安装工具）**

```bash
# 打开命令提示符（Win+R → 输入 cmd → 回车）

# 进入项目目录
cd C:\Users\Administrator\WorkBuddy\20260416091656\zhiquzhi

# 初始化 Git（只需做一次）
git init

# 添加所有文件
git add .

# 提交（记录这次更改）
git commit -m "first commit"

# 关联你的 GitHub 仓库（把 XXX 换成你的 GitHub 用户名）
git remote add origin https://github.com/XXX/zhiquzhi.git

# 上传！
git push -u origin main
```

---

### 第3步：连接 Vercel 并部署

1. 打开 Vercel：https://vercel.com
2. 点击 **Add New** → **Project**
3. 选择 **Import Git Repository**
4. 找到你的 `zhiquzhi` 仓库，点击 **Import**
5. 配置项目（保持默认即可）：
   - **Framework Preset**: Vite
   - **Root Directory**: `.`（或保持默认）
   - **Build Command**: `npm run build`
   - **Output Directory**: `dist`
6. 点击 **Deploy** ⏳

**等待 1-2 分钟...**

7. 🎉 看到 **Congratulations!** 页面了吗？
8. 点击 **Visit** 查看你的网站！

---

### 第4步：绑定自定义域名（可选）

> 如果你已经有域名，可以绑定。没有域名也能用！

1. Vercel 项目页面 → **Settings** → **Domains**
2. 输入你的域名（如 `zhiquzhi.com`）
3. 按提示在域名服务商添加 DNS 记录
4. 等待生效（通常 5-30 分钟）

## 🅱️ 方案二：Netlify 部署（同样简单）

### 第1步：注册账号

1. 打开：**https://app.netlify.com**
2. 点击 **Sign up**
3. 选择 **GitHub** 登录

### 第2步：上传代码

1. 点击 **Sites** → **Add new site** → **Deploy manually**
2. 把 `zhiquzhi` 项目里的 **`dist`** 文件夹（不是整个项目）拖进去
3. 自动部署！🎉

## 🅾️ 方案三：腾讯云/阿里云部署（国内访问快）

### 前提条件
- 需要国内服务器（腾讯云/阿里云）
- 需要域名 + **ICP 备案**

### 第1步：构建项目

1. 安装 Node.js（见下方安装教程）
2. 打开命令提示符
3. 执行：

```bash
cd C:\Users\Administrator\WorkBuddy\20260416091656\zhiquzhi
npm install
npm run build
```

4. 等待完成，生成 **`dist`** 文件夹

### 第2步：上传到腾讯云 COS

1. 登录腾讯云控制台：https://console.cloud.tencent.com
2. 搜索 **COS**（对象存储）
3. 创建存储桶：
   - 名称：`zhiquzhi`
   - 地域：选择离你近的
   - 访问权限：公有读私有写
4. 上传 `dist` 文件夹内的**所有文件**
5. 开启 **静态网站功能**：
   - 基础配置 → 静态网站 → 开启
   - 首页文档：`index.html`
   - 错误文档：`index.html`

### 第3步：配置 CDN（加速访问）

1. 开通 CDN 服务
2. 添加加速域名
3. 源站类型选择 COS
4. 配置 CNAME 解析

## 🔧 附录：Node.js 安装教程

> Node.js 是运行 JavaScript 代码的环境，部署前端项目必须安装

## ❓ 常见问题解答

### Q1: 部署后刷新页面显示 404？

**正常现象！** 这是 SPA 路由的问题。

**解决方法：**
- Vercel/Netlify 会自动处理，无需操作
- 如果用自己服务器，需要配置 Nginx（参考 DEPLOY.md）

### Q2: 微信/QQ 内打不开网站？

**微信/QQ 内置浏览器兼容性差**，这是正常的。

**解决方法：**
1. 复制网站链接
2. 打开手机浏览器（Safari/Chrome）粘贴访问

### Q3: 如何更新网站内容？

**更新代码后会自动部署：**

```bash
# 修改代码后
git add .
git commit -m "更新内容"
git push
# Vercel 会自动检测并重新部署！🎉
```

### Q4: 访问很慢怎么办？

**原因：** 服务器在海外

**解决方法：**
- 方案一：使用国内 CDN 加速
- 方案二：迁移到腾讯云/阿里云（需要备案）
- 方案三：使用香港/新加坡服务器

### Q5: 想加后台管理功能？

**推荐使用 Supabase（免费）：**

1. 注册：https://supabase.com
2. 创建项目
3. 参考 `SUPABASE_INTEGRATION.md` 文档
4. 连接后端，添加数据库、用户系统等
